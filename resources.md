# Resources

* [Anki](https://apps.ankiweb.net/): My favorite way to brute force learn anything. There are plenty of Anki decks with the top 1,000 words used in any language & more. 

### Modern Greek
* [Language Transfer](https://www.languagetransfer.org/courses#greek): An audio course for learning modern Greek available for free on [Soundcloud](https://soundcloud.com/languagetransfer/sets/complete-greek-more-audios), [YouTube](https://www.youtube.com/playlist?list=PLeA5t3dWTWvtWkl4oOV8J9SMB7L9N9Ogt), as a [zip file](https://downloads.languagetransfer.org/greek/greek.zip) or as part of the Language Transfer mobile app, available from the [App Store](https://apps.apple.com/us/app/language-transfer/id1519599004) or [Play Store](https://play.google.com/store/apps/details?id=org.languagetransfer&hl=en-US).
    * I have included a version of the LT course in this repository, which is identical to the version published in 2020, but with all of the audio files having been normalized to -1db. (The original mastered audio was annoyingly quiet)
* [Language Transfer Greek Quizlet Deck](https://quizlet.com/765856904/language-transfer-greek-flash-cards/): A set of flash-cards created by [Samuel Ellis](https://quizlet.com/user/samuel_ellis1996/sets) that tracks pretty closely with the vocabulary introduced throughout the Language Transfer series.
* [Glosbe](https://glosbe.com/el/en): A Polish dictionary website with thousands of language to language dictionaries.

### Typing Greek
* [Typing Point](https://www.typingpoint.com/greek/greek-typing): A classic *typing tutor* style website with touch typing lessons for Greek.
* [Monkey Type](https://monkeytype.com): A feature-rich multi-language typing test application that can be used for further practice.
